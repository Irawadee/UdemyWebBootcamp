HTML & CSS Basics Summary

HTML (HyperText Markup Language) Summary

defines the structure of the webpage and annotates the content to tell the browser what to display and provide extra meaning.

  • HTML adds semantic meaning to your content
  • HTML elements are typically made up of an opening tag
  • One key characteristic of HTML is that you can nest elements into each other
  • Elements for presenting and describing your page content

Learn more about HTML on MDN

CSS (Cascading Style Sheet) Summary

CSS (Cascading Style Sheets) is a language that allows you to define style definitions for your HTML document.

  • CSS properties which you can define
  • CSS rule which also a CSS selector that defines for which HTML elements your styles should be applied
  • You can really style (almost) anything with CSS
  • When working with CSS it's important to understand that HTML elements have a so-called "box model
